# Documentation

This folder contains supporting documentation for The AI Startup Game.

## Required Files

The following files are referenced in the main README and should be added to this folder:

### AI_Startup_Game_Teaching_Guide.pdf
Comprehensive guide for educators including:
- Learning objectives and outcomes
- Lesson plans for various course lengths (single session, multi-week, full course)
- Discussion questions and debriefing guidance
- Assessment ideas and rubrics
- Tips for facilitating gameplay

### Quick_Start_Guide.pdf
One-page handout for students covering:
- How to access the game
- Basic gameplay overview
- Key objectives and mechanics
- Tips for first-time players

### screenshot.png
A screenshot of the game for the README. Recommended specifications:
- Resolution: 1200x800 pixels or similar
- Format: PNG
- Content: Main gameplay screen showing key interface elements

## Adding Documentation

When adding documentation to this folder:
1. Use clear, descriptive filenames
2. Keep PDFs reasonably sized for web download
3. Update the main README if adding new documents
4. Consider accessibility (searchable PDFs, alt text for images)
